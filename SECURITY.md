# Security

Ultimate Web Fetch is designed for authorized fetching, scraping, screenshots,
PDF capture, and media downloads.

## Safe Use

- Only fetch or download content you are allowed to access.
- Do not commit `.env`, cookies, browser profiles, downloaded media, cache
  directories, screenshots that contain private data, or API keys.
- Use `--cookies-from-browser` only for accounts and content you control or
  have permission to access.
- Respect robots.txt unless you have a clear authorization reason to override
  it.

## Reporting

Please report security issues privately to the repository owner rather than
opening a public issue with exploit details.
